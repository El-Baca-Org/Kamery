# QamarSol - Dual Calendar Birthday Intelligence

An intelligent birthday tracker supporting both Gregorian and Hijri calendars with automatic conversion, dual countdowns, and multilingual support.

View your app in AI Studio: https://ai.studio/apps/drive/1342f1lABST4YQqLSwvKdIMWvNfBDCm5v

## Features
- **Dual Calendar System:** Tracks birthdays in both Gregorian and Hijri calendars simultaneously.
- **Automatic Conversion:** Seamlessly converts between Gregorian and Hijri dates.
- **Multilingual Support:** Available in Turkish, English, Ottoman Turkish, Arabic, and Persian.
- **Theme Support:** Switch between Light, Dark, or System Default modes.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key (if needed).
3. Run the app:
   `npm run dev`

# QamarSol - Çift Takvimli Doğum Günü Zekası

Hem Miladi hem de Hicri takvimleri destekleyen, otomatik dönüştürme, çift geri sayım ve çoklu dil desteği sunan akıllı bir doğum günü takip uygulaması.

Uygulamanızı AI Studio'da görüntüleyin: https://ai.studio/apps/drive/1342f1lABST4YQqLSwvKdIMWvNfBDCm5v

## Özellikler
- **Çift Takvim Sistemi:** Doğum günlerini hem Miladi hem de Hicri takvimlerde eş zamanlı olarak takip eder.
- **Otomatik Dönüştürme:** Miladi ve Hicri tarihler arasında sorunsuzca dönüşüm sağlar.
